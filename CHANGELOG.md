# Changelog

## 0.1.0 - 2026-04-30

- Launched production-ready MVP on Vercel.
- Added local JSON housing program data for Los Angeles 90029, Burbank, Glendale, Pasadena, LA County, and statewide programs.
- Added search, city filtering, category filtering, and distance tracking.
- Added local-only eligibility engine for household size and annual income checks.
- Added agency map links, dark mode, and PWA support.
- Added GitHub Actions CI, README badges, and production documentation.
