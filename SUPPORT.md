# Support

Called LA Home is an informational portfolio project and does not provide legal, financial, or housing advice.

For program eligibility, deadlines, or applications, use the official agency links in [docs/SOURCES.md](docs/SOURCES.md).

For project issues:

- Open a bug report for reproducible app problems.
- Open a feature request for focused improvements.
- Avoid sharing personal income, immigration, or housing details in public issues.
